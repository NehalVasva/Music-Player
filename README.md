# 🎵 Music Player

A responsive music player built using HTML, CSS, and JavaScript.

## Features
- Play/Pause music
- Next/Previous song
- Progress bar
- Volume control
- Playlist support
- Autoplay next song
- Glassmorphism UI
- Background image support

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download the project.
2. Add songs to the music folder.
3. Add background image to the images folder.
4. Open index.html in the browser.
